核心机制

  1. 工具描述重置 — 每10轮重置 client.last_tools = ''，防止上下文膨胀
  2. 退出条件判断 — should_exit / 空 next_prompt / 未知工具
  3. 消息历史管理 — 仅保留最新消息，历史通过 Session 管理

def try_call_generator(func, *args, **kwargs):
    ret = func(*args, **kwargs)
