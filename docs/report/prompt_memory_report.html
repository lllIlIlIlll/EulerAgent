🔬 关键洞察 Insights

🗂️ 四层记忆架构

L1(Lightweight Index) → L2(Facts) → L3(SOPs/Tools) → L4(Raw Sessions)。每层职责分明，变更时逐层同步，避免信息冗余。

🔄 严格双语同步

6个文件全部中英双语配对（sys_prompt / global_mem_insight_template / insight_fixed_structure），行数一致或相近，确保多语言环境下行为统一。

🛡️ 安全边界设计

密钥文件"仅引用不读取"、memory/ 文件"只能 patch 不能直接覆盖"、改自身源码"先请示"——三层防护防止 Agent 误操作核心资产。

📊 失败升级三阶段

1次→读错误，2次→探测环境，3次→深度分析或询问用户。配合"限制失败半径"原则，确保每次失败都有明确学习过程。

🧩 工具能力网格

15个 L3 工具 SOP + 5大能力域（Browser/键鼠/定时/自主/手机），构建了完整的能力地图，任何复杂任务都能快速定位到对应工具链。

⚖️ 架构一致性

sys_prompt（角色层） + insight_fixed_structure（架构层） + global_mem_insight_template（操作层），三层职责清晰，构成完整的 Agent 运行时上下文注入体系。
